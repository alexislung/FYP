
=== EasyJob Website Demo Instructions ===

To show this website to your teacher:

1. First run "set_env.bat" once and enter:
   - DEEPSEEK_API_KEY
   - DATABASE_URL
   - ALLOWED_ORIGINS (or press Enter for default)
2. Close that window after setup is complete.
3. Double-click "run_website.bat".
4. A black window will appear (do not close it).
5. Your default web browser will automatically open the website.
6. When finished, you can close the black window.

Note: This runs the website locally on your computer. No internet required.
